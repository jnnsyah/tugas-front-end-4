Menjalankan di Localhost

Panduan penggelaran (deployment) lokal untuk lingkungan pengembang (Development Environment).

  1. Kloning Repositori: Lakukan pengambilan repotory publik melalui protokol Git. Pastikan menargetkan *branch* utama. 
    git clone https://github.com/lab-tkj-nesaga/labflow.git
cd labflow
git checkout main
  2. Pemasangan Dependensi: Sistem LabFlow dikonfigurasi melalui paket manajer NPM standar. 
    npm install
  3. Inisiasi Variabel Lingkungan: Tentukan alamat URL pangkalan data dan parameter rahasia. 
    cp .env.example .env

    (Catatan: Buka file .env dan sesuaikan properti akses PostgreSQL)

  4. Migrasi Skema: Terapkan dan letakkan entitas tabel Prisma ORM ke dalam basis data aktif. 
    npx prisma db push
npx prisma generate
  5. Jalankan Kompilator: Eksekusi mesin server port lokal. 
    npm run dev
