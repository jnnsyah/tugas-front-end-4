Pengenalan LabFlow

LabFlow adalah sistem manajemen lab untuk jurusan TKJ di SMK Negeri 1 Gantiwarno. Sistem ini dirancang sebagai solusi pengganti distribusi Lembar Kerja Peserta Didik (LKPD) yang biasanya dikirim lewat grup WhatsApp dalam bentuk PDF yang rawan hilang.

Secara teknis, LabFlow dibangun secara ringan menggunakan Next.js dan backend Hono RPC, serta menempatkan penyimpanan file langsung di jaringan LAN lokal sekolah.

Aplikasi ini membantu siswa melihat "big picture" pembelajaran mereka melalui Visualisasi Learning Path, mengumpulkan laporan praktik per sesi dengan simpel (tanpa format rumit), dan memungkinkan guru menyusun modul digital secara mandiri melalui antarmuka drag-and-drop yang rapi.

Stack Teknologi

  • Next.js Framework
  • Hono.js API
  • Prisma ORM
